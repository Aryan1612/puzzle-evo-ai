About the Project

This research project combines educational motivation with cutting-edge AI research, bridging Large Language Model (LLM) reasoning with evolutionary program synthesis to create an automated puzzle assistant for educational settings.

Core Innovation: We address both correctness and explanation gaps in puzzle-solving AI by grounding solutions in formal representations (PDDL/DSL) while maintaining the flexibility and natural language capabilities of LLMs.

Why This Matters

üß© Puzzles Drive Learning

Spatial and logical puzzles correlate with better performance in STEM skills and abstract reasoning. UNICEF strongly advocates for 'Learning through Play', identifying puzzle-solving as critical for developing foundational cognitive skills.

ü§ñ The LLM Challenge

While Large Language Models can generate puzzles and explanations, they suffer from critical weaknesses:

  • Lack of correctness guarantees in generated solutions
  • Systematic failures in multi-step planning and algorithmic reasoning
  • Compilation and execution errors in structured outputs

üß¨ Evolutionary Synthesis Solution

Program synthesis offers a pathway to correctness by grounding solutions in formal representations. Evolutionary synthesis enables iterative refinement where candidate programs evolve via mutation and crossover guided by correctness and feedback.

Technical Approach

1. Dataset & Benchmark

Aggregate existing puzzle datasets (PuzzleVQA, AlgoPuzzleVQA) and our PDDL puzzle benchmark (~9.5K problems). Annotate with multiple solution paths and stepwise hints.

2. Hybrid Solver Pipeline

LLMs propose high-level solutions ‚Üí Evolutionary program synthesis refines candidates ‚Üí Symbolic planners verify and execute solutions.

3. Self-Improvement Loop

Use failed attempts and student interaction traces to refine the generator and search policy, following recent self-improving synthesis paradigms.

4. Educational Interface

Student-facing GUI for Grades 1-8 with puzzle solving, hint requests, multiple strategies, and logging of attempts for iterative improvement.

Project Goals

Primary Objective

A rigorous, publishable research paper that proposes a method combining evolutionary program synthesis with LLM reasoning, introduces an annotated benchmark, demonstrates gains over baselines, and releases dataset and code for reproducibility.

Secondary Objective

Lightweight Prototype Platform

Web-based GUI for student interaction, hint generation, and data collection

Pilot Studies

Initial ablation-style studies with local-language deployment in classrooms

Open Resources

Public release of datasets, code, and evaluation tools
