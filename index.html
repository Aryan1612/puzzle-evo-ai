Project Overview

This research project combines educational motivation with cutting-edge AI research, bridging Large Language Model reasoning with evolutionary program synthesis to create an automated puzzle assistant for educational settings.

Why This Matters

🧩 Puzzles Drive Learning

Spatial and logical puzzles correlate with better STEM performance. UNICEF advocates for 'Learning through Play' as critical for cognitive development.

🤖 The LLM Challenge

LLMs lack correctness guarantees, exhibit systematic failures in multi-step planning, and produce compilation errors in structured outputs.

🧬 Evolutionary Solution

Program synthesis grounds solutions in formal representations. Evolutionary refinement enables iterative improvement via mutation and crossover.

🌏 Local Impact

Pilot studies in Hindi and Bengali with real classrooms, generating valuable data and measurable educational outcomes.

Core Innovation

Hybrid solver pipeline combines LLM flexibility with formal correctness guarantees through evolutionary program synthesis.

Self-Improvement

Failed attempts and student traces refine the generator, creating a learning feedback loop.

Educational Interface

Student-facing GUI for Grades 1-8 with adaptive hints, multiple strategies, and interaction logging.

12-Month Timeline

Months 1–3
Foundation Phase
Dataset curation (PuzzleVQA, AlgoPuzzleVQA, PDDL benchmark) and baseline reproduction.
Months 4–6
Core Development
Implement LLM + symbolic + evolutionary search pipeline. Initial experiments on simple puzzles.
Months 7–9
Extension Phase
Extend to stochastic and image-based puzzles. Develop hint generation module and human evaluation setup.
Months 10–12
Finalization
Pilot GUI deployment, finalize experiments, write and submit paper. Public code/data release.

Foundation Work

Our prior work constructed a benchmark of ~9.5K PDDL problems across six puzzle types with systematic ablations using natural language, symbolic, and hybrid approaches. Currently under review, this work provides both datasets and insights for scaling into educational applications.
