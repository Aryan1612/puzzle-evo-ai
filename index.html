Towards rigorous reasoning and educational impact

Combining evolutionary program synthesis with language-model reasoning to build verified, multimodal puzzle-solving assistants and a lightweight student-facing platform for Grades 1-8.

Primary objective

Publish a rigorous method combining evolutionary program synthesis with LLM reasoning and release dataset & code.

Secondary objective

Lightweight prototype for student use, logging attempts and pedagogically-structured hints.

Motivation

Puzzle-based learning improves spatial and logical reasoning. Yet adoption in classrooms is limited by lack of localized content, adaptive feedback and teacher tooling. Our project builds an automated puzzle assistant that generates verified strategies, provides structured hints, and supports multilingual deployment.

Approach

  • Represent puzzles using compact DSL/PDDL; annotate multiple valid solution paths and hint steps.
  • LLMs propose high-level solutions; evolutionary program synthesis refines candidates; symbolic interpreters verify them.
  • Self-improvement loop using student traces to refine generators and search operators.

üéØ Measuring Success: Core Metrics

  • Solution Correctness: Every solution must be 100% executable and formally verified.
  • Solution Diversity: The system must find multiple, distinct valid solution paths.
  • Hint Usefulness: The quality of generated hints will be rated by human evaluators.
  • Generalization Power: We will test performance on unseen puzzle types to measure true reasoning.
